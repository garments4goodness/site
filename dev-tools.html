Maintenance Console

Update site data without touching HTML.

Edit chapters and blog listings, then download or copy the generated JSON. Replace the existing file in assets/data/; do not upload new root-level JSON files.

How To Use

  1. Edit the fields below.
  2. Leave a link blank if that chapter should not show the button.
  3. Click Download chapters.json or copy the output.
  4. In GitHub, open assets/data/chapters.json, click edit, replace the old contents, and commit.
  5. For blog cards, replace assets/data/articles.json the same way.
  6. Do not replace chapters.html and do not upload a new file named chapters.json at the repo root.
  7. Preview chapters.html or blog.html before publishing.

This page is public if someone knows the URL. Do not put private notes, passwords, or sensitive data here.
